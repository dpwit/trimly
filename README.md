# trimly
Barbers app for booking appointments

# 1. Create project
npx create-expo-app trimly
cd trimly

# 2. Replace files with the code from ChatGPT

# 3. Install dependencies
npm install firebase @react-navigation/native @react-navigation/stack

# 4. Start app
npx expo start

Then open it in your phone via Expo Go (iOS/Android).
